# Виджет погоды

 для установки скачайте репозиторий
```bash
git clone https://github.com/andrew1k/sono
```

Установите зависимости:
```bash
# npm
npm install
```
Или
```bash
# yarn
yarn install
```

## И запустите проект

Сервер должен запуститься по адресу `http://localhost:3000`:

```bash
# npm
npm run dev
```
Или
```bash
# yarn
yarn dev
```
